## Hacker News Pitch

**Title:** SRT Player - Free offline audio player with waveform visualization

**Body:**
```
I built a simple audio player for subtitle editing that runs entirely in the browser - no server, no API, no internet required.

Features:
- Waveform visualization (wavesurfer.js)
- SRT subtitle sync - subtitles appear in real-time as audio plays
- Click any subtitle to jump to that timestamp
- Playback speed control (0.5x to 2x)
- Works offline after first load

Background: I needed a simple tool to review subtitle timing accuracy without relying on online services. Most SRT editors require API keys or cloud processing. This one just works.

Tech: Single HTML file with bundled wavesurfer.js (~40KB). Drop it in a browser or run locally.

Inspired by voicesrt (github.com/JFK/voicesrt) - an amazing full-featured SRT editor if you need transcription + editing + YouTube tools.

Try it: github.com/arthiccc/srtplayer

Would love feedback!
```
